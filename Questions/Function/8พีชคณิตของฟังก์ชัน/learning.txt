การนำฟังก์ชันตั้งแต่ 2 ฟังก์ชันขึ้นไปมา บวก ลบ คูณ หรือหารกัน โดยมีนิยามดังนี้

(f + g)(x) = f(x) + g(x)

(f - g)(x) = f(x) - g(x)

(f · g)(x) = f(x) · g(x)

(f / g)(x) = f(x) / g(x) โดยที่ g(x) ≠ 0

โดเมนของพีชคณิตฟังก์ชัน

สำหรับ การบวก, ลบ, คูณ: โดเมนของ f₊g = โดเมนของ  f₋g = โดเมนของ  f·g = Df ∩ Dg 

สำหรับ การหาร โดเมนของ  f/g = (Df ∩ Dg) - { x | g(x) = 0 }
