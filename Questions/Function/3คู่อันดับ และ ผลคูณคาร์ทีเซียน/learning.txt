คู่อันดับ เขียนในรูป (a, b) โดยที่ a คือสมาชิกตัวหน้า และ b คือสมาชิกตัวหลัง
การเท่ากัน (a, b) = (c, d) ก็ต่อเมื่อ a = c และ b = d

ผลคูณคาร์ทีเซียน ผลคูณระหว่างเซต A และเซต B เขียนแทนด้วย A × B คือเซตของคู่อันดับทั้งหมดที่ตัวหน้ามาจาก A และตัวหลังมาจาก B
A × B = { (a, b) | a ∈ A และ b ∈ B }

**** จำนวนสมาชิก: n(A × B) = n(A) · n(B) ****
ตัวอย่าง 
A = {1,2,3} , B = {5,7,9}
n(A x B) = 3 x 3 = 9 จำนวน
