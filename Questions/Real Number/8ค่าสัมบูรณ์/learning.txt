นิยาม |x| หมายถึง ระยะห่างจากเลข 0 ถึง x บนเส้นจำนวน มีค่าเป็นบวกหรือศูนย์เสมอ
การถอดค่าสัมบูรณ์
|x| = x เมื่อ x ≥ 0
|x| = -x เมื่อ x < 0
สมบัติที่สำคัญของค่าสัมบูรณ์
 - |x| ≥ 0 เสมอสำหรับทุกจำนวนจริง x
 - |-x| = |x|
 - |x|² = x²
 - |x · y| = |x| · |y|
 - |x / y| = |x| / |y| (เมื่อ y ≠ 0)
 - |x + y| ≤ |x| + |y|
