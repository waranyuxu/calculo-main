กำหนดให้ a, b, c เป็นจำนวนจริงใดๆ
สมบัติการถ่ายทอด 
 - ถ้า a > b และ b > c แล้ว a > c
สมบัติการบวกด้วยจำนวนที่เท่ากัน 
 - ถ้า a > b แล้ว a + c > b + c
สมบัติการคูณด้วยจำนวนที่เท่ากัน
 - ถ้า a > b และ c > 0 แล้ว ac > bc (เครื่องหมายอสมการคงเดิม)
 - ถ้า a > b และ c < 0 แล้ว ac < bc (ต้องกลับเครื่องหมายอสมการเป็นตรงกันข้าม)
สมบัติการตัดออกสำหรับการบวก 
 - ถ้า a + c > b + c แล้ว a > b
สมบัติการตัดออกสำหรับการคูณ
 - ถ้า ac > bc และ c > 0 แล้ว a > b
 - ถ้า ac > bc และ c < 0 แล้ว a < b
