รูปแบบ P(x) > 0, P(x) < 0, P(x) ≥ 0, P(x) ≤ 0
ขั้นตอนการหาคำตอบ
1)จัดรูปให้อสมการฝั่งหนึ่งเป็น 0 เสมอ
2)แยกตัวประกอบพหุนามให้อยู่ในรูปกำลังหนึ่ง เช่น (x - c₁) (x - c₂) ...
3)หาค่าวิกฤต โดยจับแต่ละวงเล็บเท่ากับ 0 จะได้ x = c₁, c₂, ...
4)นำค่าวิกฤตไปพลอตลงบนเส้นจำนวน เรียงจากน้อยไปมาก (จากซ้ายไปขวา)
5)ใส่เครื่องหมาย +, -, +, - สลับกัน โดยเริ่มจากขวาสุดเป็นเครื่องหมาย + เสมอ 
6)เลือกช่วงคำตอบบนเส้นจำนวน
 - อสมการ > 0 เลือกช่วงที่เป็นบวก (+) และใช้จุดโปร่ง (ช่วงเปิด)
 - อสมการ < 0 เลือกช่วงที่เป็นลบ (-) และใช้จุดโปร่ง (ช่วงเปิด)
 - อสมการ ≥ 0 เลือกช่วงที่เป็นบวก (+) และใช้จุดทึบ (ช่วงปิด)
 - อสมการ ≤ 0 เลือกช่วงที่เป็นลบ (-) และใช้จุดทึบ (ช่วงปิด)
*** หากเป็นอสมการเศษส่วน ตัวส่วนห้ามเป็น 0 เสมอ ให้ใช้จุดโปร่งที่ค่าวิกฤตของตัวส่วน ***
