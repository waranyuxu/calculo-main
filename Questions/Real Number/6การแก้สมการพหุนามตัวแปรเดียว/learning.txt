การแก้สมการพหุนามตัวแปรเดียว
 - รูปทั่วไป aₙxⁿ + aₙ₋₁xⁿ⁻¹ + ... + a₁x + a₀ = 0 (เมื่อ aₙ ≠ 0)
สมการดีกรี 2 (ax² + bx + c = 0):
 - วิธีที่ 1: แยกตัวประกอบให้อยู่ในรูป (x - m)(x - n) = 0 จะได้ x = m, n
 - วิธีที่ 2: ใช้สูตร x = [-b ± √(b² - 4ac)] / 2a
สมการดีกรีมากกว่า 2
 - ใช้ทฤษฎีบทเศษเหลือเพื่อสุ่มหาค่า c ที่ทำให้ P(c) = 0
 - นำค่า c ไปใช้ในการหารสังเคราะห์ เพื่อลดดีกรีของสมการลง แล้วจึงแยกตัวประกอบต่อจนเสร็จ
