ลำดับเลขคณิต คือ ลำดับของตัวเลขที่แต่ละพจน์เพิ่มขึ้นหรือลดลงด้วยการบวกหรือลบค่าคงที่ที่เรียกว่า "ผลต่างร่วม" (d) ด้วยจำนวนที่เท่ากันเสมอ
ตัวอย่าง
 - ลำดับที่ 1 : 1,3,5,7,9,...
 - ลำดับที่ 2 : 12,11,10,9,...
สูตรที่สำคัญของลำดับเลขคณิต
 - การหาพจน์ทั่วไปหรือพจน์ที่ n ของลำดับเลขคณิต 
   -> aₙ = a₁ + (n-1)d
   -> aₙ = พจน์ที่ n 
   -> a₁ = พจน์ที่ 1
   -> n  = ลำดับที่ ของพจน์นั้นๆ
   -> d  = ผลต่างร่วม หาได้จาก aₙ₊₁ - aₙ = d
